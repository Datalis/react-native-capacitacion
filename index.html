La modificación es asíncrona

Incorrecto



						this.setState({
							counter: this.state.counter + this.props.increment,
						  });

Correcto


						this.setState((state, props) => ({
							counter: state.counter + props.increment
						  }));

Persistencia de Datos con el LocalStorage

  • setItem(nombre, dato) - Guardar datos

  • getItem(nombre) - Acceder a datos

  • removeItem(nombre) - Borrar un dato

  • clear(dominio) - Eliminar todos los datos
